# 📱 智投手帳 - 完整使用指南

**版本**: 1.0
**更新日期**: 2024-11-25

---

## 🎯 歡迎使用智投手帳！

智投手帳是一個雙平台財務管理工具，結合 **LINE Bot 快速記帳** 與 **網頁版詳細分析**，幫助你輕鬆管理個人財務與投資組合。

---

## 🚀 快速開始（5 分鐘上手）

### 步驟 1️⃣：加入 LINE 好友

1. 掃描 QR Code 或點擊邀請連結
2. 加入「智投手帳」LINE 官方帳號
3. Bot 會自動發送歡迎訊息

---

### 步驟 2️⃣：建立你的第一個帳戶

#### 方法一：透過網頁版（推薦）✨

1. 在 LINE Bot 輸入：`網站` 或 `帳戶`
2. 點擊「🌐 開啟網站」按鈕
3. 進入網頁後，點擊右上角「⚙️ 設定」
4. 選擇「帳戶管理」
5. 點擊「➕ 新增帳戶」

**建議建立的帳戶**：

| 帳戶類型 | 名稱範例 | 幣別 | 用途 |
|---------|---------|------|------|
| 💵 現金錢包 | 現金 | TWD | 日常支出 |
| 🏦 銀行帳戶 | 台新銀行 | TWD | 儲蓄、轉帳 |
| 🏛️ 證券戶 | 國泰證券 TWD | TWD | 台股交易 |
| 🏛️ 證券戶（選配） | Firstrade | USD | 美股交易 |

**填寫資訊**：
- **帳戶名稱**：例如「台新銀行」、「現金錢包」
- **帳戶類型**：選擇「銀行帳戶」、「證券戶」等
- **幣別**：TWD（台幣）或 USD（美金）
- **初始餘額**：輸入當前餘額（例如：10000）
- **設為預設**：勾選最常用的帳戶

#### 方法二：透過 LINE Bot

1. 輸入：`建立帳戶`
2. Bot 會引導你前往網頁版設定

---

### 步驟 3️⃣：開始記帳（三種方式）

#### 🌟 方式 1：一步式記帳（最快）

**記支出**：直接說出消費內容
```
午餐 120
咖啡 80
計程車 200
電影 350
```

**記收入**：直接說出收入內容
```
薪水 50000
獎金 10000
股息 500
```

#### ⚡ 方式 2：兩步式記帳（傳統）

**記支出**：輸入金額 → 選擇分類
```
-120        (Bot 會顯示分類選單)
→ 選擇「飲食」
```

**記收入**：輸入金額 → 選擇分類
```
+5000       (Bot 會顯示分類選單)
→ 選擇「薪資」
```

#### 💡 方式 3：智能預測（自動分類）

當你輸入金額時，Bot 會根據：
- **時間規律**（午餐時間 → 飲食）
- **金額大小**（大額 → 居住）
- **歷史習慣**（相似金額的常用分類）

自動預測分類，你只需確認即可！

**範例**：
```
輸入：-120
Bot：💡 智能預測：飲食 (金額: 120 元)
     請選擇類別或確認預測
     [🎯 飲食] [交通] [居住] ...
```

### 步驟 4️⃣：查詢股價與買賣股票

#### 📊 查詢股價

**台股**（直接輸入代碼）：
```
2330        → 台積電
0050        → 元大台灣 50
00878       → 國泰永續高股息
00919       → 群益台灣精選高息
```

**美股**（直接輸入代碼）：
```
AAPL        → Apple
TSLA        → Tesla
NVDA        → NVIDIA
```

#### 💰 買入股票

**方法 1：LINE Bot 對話式買入**
```
1. 輸入：買 2330
2. Bot 顯示現價並詢問：「請輸入股數」
3. 輸入股數：10
4. Bot 計算成本並確認
5. 確認後自動扣款並記錄
```

**方法 2：網頁版買入**（推薦，功能更完整）
1. 開啟網站 → 持股列表
2. 點擊「➕ 買入」
3. 搜尋股票代碼
4. 輸入股數、價格
5. 選擇扣款帳戶
6. 確認買入

#### 📉 賣出股票

**LINE Bot**：
```
賣 2330
→ 輸入股數
→ 自動計算損益並入帳
```

**網頁版**：
1. 點擊持股卡片
2. 選擇「賣出」
3. 輸入股數
4. 確認賣出

## 📚 進階功能

### 🔄 帳戶轉帳

**使用情境**：
- 銀行 → 證券戶（入金）
- 證券戶 → 銀行（出金）
- TWD 帳戶 → USD 帳戶（換匯）

**操作方式**（僅限網頁版）：
1. 設定 → 帳戶管理
2. 點擊「💸 轉帳」
3. 選擇來源帳戶
4. 選擇目標帳戶
5. 輸入金額（可選填手續費）
6. 確認轉帳

---

### 💹 複委託買美股

**情境**：使用台幣帳戶買美股（透過券商換匯）

**操作**：
1. 確保有 TWD 證券帳戶
2. 搜尋美股（例如：AAPL）
3. 系統會自動偵測需要換匯
4. 顯示換匯提示：
   ```
   US$150 × 32.5 = NT$4,875
   ⚠️ 實際匯率以券商為準
   ```
5. 確認後自動扣除台幣並記錄

### 智能分類規則

Bot 會根據以下規則預測分類：

**時間規則**：
- 7-9點 + 小額 → 早餐（飲食）
- 11-14點 + 小額 → 午餐（飲食）
- 17-20點 + 小額 → 晚餐（飲食）
- 7-9點 + 中額 → 通勤（交通）

**金額規則**：
- \>10,000 → 居住
- 5,000-10,000 → 購物
- 2,000-5,000 → 娛樂或購物
- 500-2,000 → 交通或娛樂
- <500 → 飲食或交通

**歷史規則**：
- 相似金額（±20%）的最常用分類

## 🛠️ 常見問題（FAQ）

### Q1: 輸入 `2330` 會被當成記帳金額嗎？

**A**: 不會！系統會優先判斷為股票代碼。
如果要記帳 2330 元，請輸入 `+2330` 或 `-2330`。

---

### Q2: 如何刪除錯誤的交易記錄？

**A**: 目前僅支援網頁版刪除：
1. 開啟收支手帳（Ledger）
2. 找到該筆交易
3. 點擊右側「🗑️ 刪除」

---

### Q3: 買股票時提示「餘額不足」怎麼辦？

**A**:
1. 檢查選擇的帳戶餘額是否足夠
2. 若不足，先轉帳入金到證券戶
3. 或者在銀行帳戶先存入資金

---

### Q4: 複委託是什麼？

**A**: 複委託是用 **台幣帳戶** 買 **美股**，券商會自動換匯。
系統支援：選擇 TWD 證券戶 → 買美股 → 自動計算換匯成本

---

### Q5: 如何查看過去的股票交易記錄？

**A**:
- **LINE Bot**：輸入 `持倉` 查看當前持股
- **網頁版**：收支手帳 → 篩選「投資」類別

---

### Q6: 智能預測不準確怎麼辦？

**A**:
1. 手動選擇正確的分類（系統會學習）
2. 使用一步式記帳直接指定分類（例如：`午餐 120`）
3. 多記帳幾次後，歷史規則會越來越準確

---

### Q7: 如何設定帳戶為預設？

**A**: 網頁版：設定 → 帳戶管理 → 點擊帳戶 → 點擊「⭐ 設為預設」
